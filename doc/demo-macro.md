## Tastenkombination:
<pre>
^ = Strg
! = Alt
+ = Shift
# = Windows Taste
</pre>

Bsp.:
`^n =  Strg + N`

## Wartezeiten:
<pre>
; = 2500ms
, = 1500ms
. = 500ms
</pre>

## Beispieltastaturen:
<pre>
{TAB}
{RIGHT}
{DOWN}
{ENTER}
</pre>
